# Инструкция по использованию PDF Compressor

## Установка расширения

1. Склонируйте репозиторий или скачайте исходный код
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Соберите расширение:
   ```bash
   npm run build
   ```
4. Откройте Chrome и перейдите на страницу `chrome://extensions/`
5. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
6. Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked)
7. Выберите папку `dist` из проекта

## Использование

1. Нажмите на иконку расширения в панели инструментов Chrome, чтобы открыть боковую панель
2. В боковой панели нажмите кнопку "Выберите PDF файл"
3. Выберите PDF файл, который хотите сжать
4. Дождитесь завершения процесса сжатия
5. Просмотрите результаты:
   - Исходный размер файла
   - Размер сжатого файла
   - Процент уменьшения размера
6. Нажмите кнопку "Скачать сжатый PDF" для загрузки результата

**Преимущество боковой панели**: Панель остается открытой при навигации между вкладками, что позволяет удобно работать с несколькими PDF файлами.

## Ограничения

- Максимальный размер файла: 50 МБ
- Поддерживаются только файлы формата PDF
- Расширение работает только с валидными PDF файлами

## Конфиденциальность

Все операции выполняются локально в вашем браузере. Файлы не отправляются на внешние серверы. Временные данные автоматически удаляются при закрытии расширения.

## Устранение неполадок

### Ошибка "Неверный тип файла"
- Убедитесь, что выбранный файл имеет расширение `.pdf`
- Проверьте, что файл не поврежден

### Ошибка "Файл слишком большой"
- Размер файла превышает 50 МБ
- Попробуйте сжать файл другими средствами перед использованием расширения

### Ошибка "Файл поврежден"
- PDF файл имеет неподдерживаемый формат или поврежден
- Попробуйте открыть файл в PDF-ридере для проверки

### Ошибка сжатия
- Попробуйте другой PDF файл
- Перезагрузите расширение
- Проверьте консоль браузера для дополнительной информации

## Технологии

- TypeScript
- pdf.js (pdfjs-dist) - для извлечения и рендеринга PDF страниц
- pdf-lib - для создания сжатых PDF файлов
- Canvas API - для сжатия изображений
- Vite (сборщик)
- Vitest (тестирование)
- fast-check (property-based тестирование)

## Как работает сжатие

Расширение использует следующий подход для эффективного сжатия PDF:

1. **Извлечение страниц**: pdf.js загружает PDF и извлекает каждую страницу
2. **Рендеринг на Canvas**: Каждая страница рендерится на HTML Canvas элемент
3. **Сжатие изображений**: Canvas API конвертирует страницу в JPEG с настраиваемым качеством (по умолчанию 70%)
4. **Создание нового PDF**: pdf-lib создает новый PDF из сжатых изображений
5. **Оптимизация**: Удаление метаданных и оптимизация структуры PDF

Этот метод особенно эффективен для PDF файлов, содержащих изображения или отсканированные документы. Типичное сжатие составляет 50-80% от исходного размера.
