## Detailed Description

**PDF Compress - Local Tool** — compress your PDF files directly in the browser.

Looking for a **PDF compress** solution that keeps your files private? This tool compresses PDF files locally on your device — no server uploads required.

**Why use this PDF compress extension?**
- 🔒 **100% Private PDF compress** — your files never leave your browser
- 📉 **Compress PDF 50-80%** — especially effective for image-heavy or scanned documents
- ⚡ **Fast PDF compress** — no upload time, instant results
- 🛠️ **Free PDF compress tool** — no limitations, open source
- 💎 **Simple PDF compress** — select file, compress, download

**How to compress PDF:**
1. Click extension icon to open the panel
2. Select your PDF file
3. Automatic compression starts
4. Download your compressed PDF

**Use cases:**
- Compress PDF for email attachments
- Compress PDF for web upload
- Compress PDF for storage optimization
- Compress scanned documents

This **PDF compress** tool uses Canvas API and JPEG compression to reduce file size while maintaining document quality. Built with pdf.js and pdf-lib.
