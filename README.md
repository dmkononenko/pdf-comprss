# PDF Compressor Chrome Extension

Chrome расширение для локального сжатия PDF файлов.

## Возможности

- ✅ Локальное сжатие PDF файлов без отправки на сервер
- ✅ Эффективное сжатие изображений в PDF (50-80% уменьшение размера)
- ✅ Использование open source библиотек (pdf.js + pdf-lib)
- ✅ Рендеринг страниц через Canvas API с JPEG сжатием
- ✅ Удаление метаданных для уменьшения размера
- ✅ Оптимизация структуры PDF
- ✅ Отображение статистики сжатия (исходный размер, сжатый размер, процент уменьшения)
- ✅ Удобный интерфейс в виде боковой панели (sidebar)
- ✅ Полная конфиденциальность - все операции выполняются локально в браузере

## Как это работает

Расширение использует комбинацию двух мощных библиотек:

1. **pdf.js** - извлекает и рендерит каждую страницу PDF
2. **Canvas API** - сжимает страницы в JPEG формат с настраиваемым качеством
3. **pdf-lib** - создает новый оптимизированный PDF из сжатых изображений

Этот подход обеспечивает значительное уменьшение размера файла, особенно для PDF с изображениями или отсканированных документов.

## Загрузка расширения в Chrome

1. Выполните сборку: `npm run build`
2. Откройте Chrome и перейдите в `chrome://extensions/`
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку `dist`
6. Нажмите на иконку расширения в панели инструментов, чтобы открыть боковую панель
