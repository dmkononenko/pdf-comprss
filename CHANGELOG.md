## [1.1.0] - 2024-12-08

### Добавлено
- Интеграция библиотеки pdfjs-dist для эффективного извлечения страниц PDF
- Реальное сжатие изображений через Canvas API с JPEG компрессией
- Параметр `imageScale` для настройки разрешения рендеринга
- Конфигурация Vite для корректной работы с pdf.js worker
- Web accessible resources для pdf.worker.mjs в manifest.json

### Изменено
- Полностью переработан алгоритм сжатия в `CompressionEngine`
- Обновлен интерфейс `CompressionOptions` с добавлением `imageScale`
- Улучшена документация в design.md с детальным описанием алгоритма
- Обновлены README.md и USAGE.md с информацией о новом подходе

### Исправлено
- Проблема с неэффективным сжатием (было ~0%, стало 50-80% для PDF с изображениями)
- Корректная настройка pdf.js worker для работы в Chrome Extension

### Технические детали

**Новый алгоритм сжатия:**
1. pdf.js загружает и извлекает страницы PDF
2. Каждая страница рендерится на Canvas элемент
3. Canvas API конвертирует страницу в JPEG с качеством 70%
4. pdf-lib создает новый PDF из сжатых изображений
5. Применяется удаление метаданных и оптимизация структуры

**Параметры по умолчанию:**
- `imageQuality: 70` - качество JPEG (0-100)
- `imageScale: 1.0` - масштаб рендеринга (1.0 = оригинальное разрешение)
- `removeMetadata: true` - удаление метаданных
- `optimizeStructure: true` - оптимизация структуры PDF
